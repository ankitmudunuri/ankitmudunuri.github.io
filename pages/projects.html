LinkedIn Job Scraper

This was a project that I challenged myself with to practice how to code quick. I gave myself an alloted time of an hour and a half to plan out the mini-project and actually create it. It also gave me practice of reading documentation, as I can proudly say I finished this project just by testing it myself and utilizing the Selenium and Google Sheets API without the need of an online guide or tutorial.

The project itself serves as a way to aggregate all jobs based on a certain job keyword search on LinkedIn.com. It allows a streamlined way to aggregate potential jobs to apply to. There are still some kinks that I need to sort out, such as making a proper UI and making it so that anyone can use it with their own Google account token.
